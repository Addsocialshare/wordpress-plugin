== Description ==
Just install this FREE plugin to enable Social Sharing on your WordPress site. This plugin is also compatible with WordPress multisite.

See the <a href="http://wordpress.org/plugins/addsocialshare/screenshot/">screenshot section</a> for visuals on social sharing features.

= Configuring the Widget =

1. Once you activate the plugin, go to the 'Widgets' section in the 'Design' or 'Appearance' menu.
2. Look for the 'addsocialshare' widget and click on the 'Add' link, or drag and drop the widget to the widget area on the right.
3. After adding the addsocialshare widget, customize it by changing the title, text show before, text show after, etc.
